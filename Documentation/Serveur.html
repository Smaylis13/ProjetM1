Documentation du serveur

Voici la page de documentation concernant le serveur.

Spécificités du serveur

La fonction du serveur est de servir de lien entre les differentes applications en faisant transiter les données et en les stockant. Il permet également de gérer l'identification avec les login et password.

Le serveur se devait d'être distant et accessible à partir de l'université. Pour celà nous avons eu recours à un VPS

Technologies

Le serveur est équipé des technologies suivante :

  • Apache
  • PHPMyAdmin
  • MySQL
  • Java 8
  • Maven

Gestion des données

Le serveur dispose d'une base de données qui stocke les informations concernant les utilisateurs, les votes et leurs résultats.
